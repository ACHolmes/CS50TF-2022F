HTML/CSS practice


A colloborative HTML/CSS exercise for some practice :)

  1. You have 10 minutes to collobaritvely re-create this page as best as possible.
  2. You may work all as one, or depending on numbers, small groups.
  3. No Javascript is required
  4. You may use inline styling, style tags or a CSS file for any CSS.
  5. You may use Google/ any resources available to you!
  6. Don't worry about replicating all of the text, you can just put placeholder text (lorem ipsum or just a placeholder word/phrase)
